# todo.asobou.vue.js
todo.asobou.vue.js

推奨ブラウザ
・Google Chrome
・Firefox
-----------------------------
すべて
→すべてのコメントが表示されます

作業
→作業中のコメントが表示されます

完了
→完了したコメントが表示されます

（No.件を表示）
→コメントの件数が表示されます
-----------------------------

-----------------------------
ID
→コメントを追加すると番号が割り振られます

コメント
→コメントを追加すると文字が表示されます

状態
→ボタンを押下すると状態が切り替わります

○編集
→指定のコメントを編集できます。
　削除後に編集する際には、必ず更新してから実施ください
　削除した際には、IDごと削除してしまうため
　編集がうまくできない可能性があります。

●削除
→ctrlを押しながらクリックすると削除できます
-----------------------------

-----------------------------
新しい作業の追加
→「コメント」に入力後に追加すると新しいコメントを作成できます
-----------------------------
